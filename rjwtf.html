Python's Name

Python wasn't named after the snake‚ÄîGuido van Rossum named it after Monty Python's Flying Circus. The language's philosophy "There's only one way to do it" is actually a joke reference to the show's humor.

JavaScript in 10 Days

Brendan Eich created JavaScript in just 10 days in 1995. It was originally called Mocha, then LiveScript, before finally becoming JavaScript to ride Java's marketing wave.

React's Virtual DOM

React's Virtual DOM was inspired by XHP, a PHP extension. Facebook engineers wanted to bring component-based UI to JavaScript, leading to one of the most popular frameworks today.

HTML's First Website

The first website ever created is still online at info.cern.ch. It was created by Tim Berners-Lee in 1991 and explains what the World Wide Web project was about.

Git's Creator

Linus Torvalds created Git in 2005 after the Linux kernel project couldn't use BitKeeper anymore. He designed it to be fast, distributed, and handle large projects like Linux.

First Computer Bug

The term "bug" comes from 1947 when Grace Hopper found an actual moth stuck in Harvard's Mark II computer. The moth was taped into the logbook with the note "First actual case of bug being found."
